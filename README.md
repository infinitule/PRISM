# Virtual Fiber Optic Neural Network

A software simulation of a photonic neural network where **every dot product is computed by a pulse of coherent light** — not a transistor.

Built from Zang et al., *"Fiber neural networks for intelligent optical fiber communication signal processing"* (iOptics 2025) and ITU-T G.652 single-mode fiber physics.

---

## Requirements

- Python 3.9+
- NumPy ≥ 1.24
- No GPU. No PyTorch. No TensorFlow.

```bash
pip install -r requirements.txt
```

---

## Run it

### Recursive self-improving engine
```bash
cd fiber_agi
python3 main.py
```

Runs 6 generations of recursive development. Each generation either deepens (new fiber ring loop) or widens (expands the narrowest hidden layer), warm-started from the previous generation's weights. Stops when fiber accuracy exceeds 99%.

### Four production telecom applications
```bash
python3 use_cases.py
```

| Use Case | Task | Network |
|---|---|---|
| UC1 Multi-Impairment Classifier | Diagnose fiber link health: Normal / CD / PMD / OSNR / Nonlinear | [8,20,10,5] |
| UC2 Adaptive Modulation Controller | Pick best MCS: OOK → QAM-64 | [8,24,12,6] |
| UC3 OTDR Fault Diagnosis Engine | Identify fault from OTDR trace: Normal / Connector / Splice / Break / MacroBend | [8,20,10,5] |
| UC4 DCI Link State Classifier | 400ZR/ZR+ transceiver health: Optimal / Degraded / Marginal / Critical / Failed | [8,20,10,5] |

All four run on the same physical optical substrate — 1550 nm laser, IQ modulator, SMF G.652, EDFA, DCF, homodyne receiver. Only the trained weights change.

## How it works

### The optical neuron
```
Laser (1550 nm CW)
  → IQ modulator:  amplitude = |w|,  phase = 0 if w≥0, π if w<0
  → IQ modulator:  amplitude = |x|,  phase = 0 if x≥0, π if x<0
  → DCF coherent sum:  E = Σ wᵢxᵢ · e^(jφ)
  → 90°-hybrid + balanced photodetector
  → Output:  I ∝ Re(E_sig · E_LO*) = Σ wᵢxᵢ
```

This is a **single-pass exact dot product** using coherent IQ detection. No approximation. Zero theory-to-fiber gap.

### Physical substrate (ITU-T G.652)
| Parameter | Value |
|---|---|
| Wavelength | 1550 nm (C-band) |
| Fiber | SMF G.652, NA=0.1235, V=2.252 (single-mode) |
| Attenuation | 0.2 dB/km |
| GVD | β₂ = −21.7 ps²/km |
| EDFA | G=20 dB, NF=4 dB |
| DCF | −100 ps²/nm·km, 28.5 km |
| WDM spacing | 0.8 nm (100 GHz DWDM grid) |
| Detector | Responsivity R=0.9 A/W |

### The recursive meta-prompt
Weights are not random — they are generated by a 3-level hierarchy:

```
Level 0:  P₀ ∈ ℝ⁶⁴  (Fibonacci seed)
Level 1:  Ψ(P₀) → (M_W1, M_W2)  via fixed random projection
Level 2:  MetaGen(Φ, P) → W  (outer product projection)
```

The seed generates the generator that generates the weights. Across recursive generations, a memory vector φ₁ accumulates context — the network remembers its own history.

## Modifying it

**Bigger network:**
```python
# main.py
initial_layer_sizes = [8, 32, 16, 5]
```

**More recursive generations:**
```python
# main.py
engine.run(max_gen=10, target_acc=0.999)
```

**Add a new use case:**
Copy any `run_ucN` block in `use_cases.py`. Define `_myuc_sample(rng, label)` with label-first feature generation (generate features conditioned on class, not the other way around). Call it from `main()`.

**Swap to intensity-mode (V1, 4-pass decomposition):**
Replace `CoherentFiberNetwork` with `FiberNeuralNetwork` from `fiber_nn.py`.
